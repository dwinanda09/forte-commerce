Logo

The FC mark is a flowing f→c lettermark. The f uses an infinity-loop stroke that flows directly into the open c. Never stretch, recolor, or add drop-shadows.

Logo Usage Rules

  • Minimum size: 24px height — below this the stroke detail is lost.
  • Maintain clear space equal to the logo height on all sides.
  • On light backgrounds: use original black mark.
  • On dark or teal backgrounds: use white (CSS filter: invert(1) or white SVG variant).
  • Never add outline, glow, or drop-shadow to the mark itself.

Design Rules

  • Teal (#01796F) is the only action color — no secondary accents.
  • Animate transform and opacity only — never layout properties.
  • Every interactive element needs a :focus-visible ring in teal.
  • Cards: white bg + shadow-card. Page: surface (#F8FAFB).
  • No decorative gradients — gradient only used inside product image placeholder areas.
  • Max two font weights: 400 (regular) and 600/700 (semibold/bold).
  • Out-of-stock: product remains visible, button disabled + opacity 40%, price in steel.
  • Status badges use colored backgrounds, not just colored text (accessibility).
